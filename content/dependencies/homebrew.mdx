# Homebrew

## What is Homebrew?

Homebrew is a package manager for macOS that allows us to easily install software and manage dependencies. More info can be found on the [official Homebrew website](https://brew.sh/).

## Homebrew Installation

Homebrew has two installation methods. In this guide we use the terminal, but you can also use the `.pkg` installer from the [Homebrew repository](https://github.com/Homebrew/brew/releases/latest).

Open your terminal and run:

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Then add Homebrew to your path:

```sh
echo >> ~/.zprofile
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
```

Verify the installation:

```sh
brew doctor
```
