# Dependencies

For our initial setup, we will need to install a few dependencies. These dependencies will help us manage the project efficently, and save time over the course of this guide. All of these installations are done in the terminal. Make sure to install the following dependencies in order:

1. **Homebrew** - Homebrew is a package manager for macOS that allows you to easily install software and manage dependencies. You can install Homebrew by running the following command in your terminal:

   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
